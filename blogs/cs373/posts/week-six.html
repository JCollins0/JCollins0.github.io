What's my pick-of-the-week or tip-of-the-week?

My pick-of-the-week is WinSCP. WinSCP is a file transfer tool that uses SSH connections and allows you to access the remote file system. It is very similar to the sshfs linux package. The default editor that WinSCP comes with is similar to notepad, but you can change that to allow you to open the files in any editor that you want. For me that's Atom and it works great; I can have syntax highlighting and formatting and then Ctrl+S to save it on the remote server.
